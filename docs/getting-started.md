# 快速开始

欢迎来到 RIME · 雾凇服务器！本页面将帮助你快速了解我们的服务器特色和选择适合的区服。

## 🎯 服务器概览

RIME · 雾凇是一个专注于科技整合包的 Minecraft 服务器，我们提供两个不同特色的区服：

## 第一区服：GregTech Odyssey

第一区服基于国创整合包 **[GTO] 格雷科技：新征程 (GregTech Odyssey)**，主打高难度长线科技发展。

- **整合包类型**：以 GregTech 为核心的魔改科技整合包，辅以魔法与冒险内容
- **玩法节奏**：从 LV 一路推进到 MAX，完整体验 GT 机器与多方块流程
- **性能与优化**：使用自研核心对 AE 并行计算、GT 机器批处理、多模组兼容等做了大量优化，提升中后期服务器与客户端性能
- **自动化支持**：提供多种样板总成、大型多方块机器与自动化方案，方便玩家搭建大规模工厂
- **游戏版本**：Minecraft **1.20.1**（Forge 服务器）
- **推荐 Java**：64 位 **JDK 21**（如客户端自带，请优先使用自带）
- **推荐配置**：至少分配 2G 内存，推荐 4G 及以上以获得更流畅体验

进入一服后，可以按照任务线循序渐进推进科技，适合喜欢硬核进度与工厂规划的玩家。

## 第二区服：ATM10 · To The Sky

第二区服采用整合包 ATM10 · To The Sky，属于 All The Mods 系列的空岛玩法，提供现代化的高模组整合体验。

- **核心玩法**：在空岛起步，通过资源生成器、压缩方块与各类机械逐步拓展领地
- **任务与引导**：任务书提供阶段性目标，帮助玩家快速解锁重要模组与关键机器
- **科技深度**：涵盖 Create、Mekanism、AllTheModium 等模组，支持多线路自动化与材料升级
- **游戏版本**：Minecraft **1.20.1**
- **推荐 Java**：64 位 **JDK 21**（如客户端自带，请优先使用自带）
- **适合人群**：喜欢空岛成长、愿意尝试多模组联动与自动化设计、想与朋友一起推进任务链的玩家

进入二服后，建议先按任务书指引搭建基础资源生成，再逐步解锁大型科技模组打造空岛工厂。
